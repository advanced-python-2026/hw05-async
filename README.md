Полный текст задания: https://advanced-python.ru/advanced-python/homework/hw05

## Быстрый старт

1. **Заполни `STUDENT.md`** — впиши ФИО точно как в ведомости (от этого зависит вариант).
2. **Установи зависимости:**
   ```bash
   uv sync
   ```
3. **Посмотри свой вариант:**
   ```bash
   uv run pytest -v --co
   ```
4. **Реализуй задания** в `hw05/fixed_async.py`, `hw05/downloader.py` и `hw05/pipeline.py`.
5. **Проверь локально:**
   ```bash
   uv run ruff check .
   uv run ruff format --check .
   uv run pytest -v
   ```
6. **Запуш в main** — CI проверит автоматически.

## Задания

### 5.1 — Найди и исправь блокирующие операции (2 балла)

В файле `src/broken_async.py` находится асинхронный сервис обработки данных.
В нём скрыто несколько блокирующих операций, которые нарушают работу event loop.

1. Скопируй `src/broken_async.py` в `hw05/fixed_async.py`.
2. Найди и исправь все блокирующие вызовы.
3. Для каждого исправления добавь комментарий с объяснением:
   - Что именно блокирует event loop
   - Почему это проблема
   - Как исправление решает проблему

### 5.2 — Асинхронная загрузка с семафором (2 балла)

Реализуй функцию `download_all(urls, dest_dir, max_concurrent=5)` в `hw05/downloader.py`:
- Асинхронная загрузка файлов по URL
- Ограничение параллельности через `asyncio.Semaphore`
- Корректная обработка ошибок (HTTP-ошибки, таймауты)
- Прогресс-бар через `tqdm`
- Возвращает `dict[str, bool]` — URL -> успех/неудача

### 5.3 — Асинхронный пайплайн с генераторами (2 балла)

Реализуй асинхронный пайплайн обработки данных в `hw05/pipeline.py`:
- `read_chunks(path, chunk_size)` — читает файл чанками
- `parse_lines(chunks)` — собирает строки из чанков
- `filter_lines(lines, pattern)` — фильтрует строки по паттерну
- `batch(items, size)` — группирует элементы в батчи

Формат данных зависит от варианта (определяется по ФИО).
